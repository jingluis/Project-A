Instruccions per compilar:
make all

Instruccions per executar:
./graf

Descripció graf.cc:
Fitxer que conté la funció main, funcions bàsiques com DFS i
calcular components connexos i fer tests estadístics.

Descripció random_graphs.cc random_graphs.hh:
Fitxer que conté les funcions per crear binomial random graphs i
random geometric graphs.

Descripció utilc.cc utils.hh:
Fitxer que inclou les llibreries que necessita.

Descripció edge_graphics.R:
Script R per donar gràfiques del test de la relació entre
la probabilitat que el graf sigui connex i el nombre d'arestes.

Descripció GRAPHICS.R:
Script R per donar gràfiques del test de la relació entre
la probabilitat que el graf sigui connex/nombre de components connexos/mida màxima del component connex
i els paràmetres p/r
